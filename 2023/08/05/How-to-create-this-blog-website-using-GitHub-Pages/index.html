How to create this blog website using GitHub Pages

搭建博客思路

  1. 利用Github Pages作为博客发布站点，存储博客内容并部署为html网页，便于公众访问。
  2. 利用hexo创建并管理博客。博主只需要提供markdown格式的内容文档，利用hexo可以自动形成样式又没的网页内容，并且具有丰富的主题供可以选择。
  3. 选定hexo主题，修改样式，创建博客存储在Github仓库中，并利用Github Pages部署为网页。

创建Github 仓库

1，注册GitHub账号，注意设置username

2，新建仓库，并将仓库命名为指定格式

仓库名字必须为username.github.io，这也是博客主页的访问网址。

3，将仓库部署为GitHub Pages

部署过程只需要一次操作，之后Github会自动从仓库中获取内容，并发布为网页。
